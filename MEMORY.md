**真实工程 (非骨架)**:

#### 1. LICM (Loop Invariant Code Motion) 真实实现
位置: `原生编译器/平台/代码生成/x86_64/代码生成器.nova:3224-3389`
实现:
- `_LICM_是计数调用(节点)` — 识别 `计数(X)` 调用模式
- `_LICM_取数组名(节点)` — 提取数组变量名
- `_LICM_节点修改变量(节点, 变量名)` — 检测赋值/追加/删除/字典设/索引赋值/非纯函数调用
- `_LICM_体修改变量(体, 变量名)` — 递归深入嵌套循环/条件
- 主流程集成到 `编译_处理_节_当` 开头:
  * 识别 `变量 < 计数(X)` 模式
  * 若 X 循环内不变, 分配临时栈变量, 循环头前预计算
  * AST 重写右操作数为临时变量引用
- 保守策略: 白名单纯读函数 (计数/字典取/类型/打印...), 未知函数视为变异
