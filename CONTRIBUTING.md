## How to Contribute / 如何贡献

1. **Fork** this repository / 复刻本仓库
2. Create a **feature branch** / 创建功能分支
3. Make your changes with **clear commit messages** / 提交清晰的变更说明
4. Submit a **Pull Request** / 提交拉取请求
5. Ensure your code follows the project's **Chinese-native conventions**
   确保代码遵循项目的中文原生约定

## Code Conventions / 代码规范

- All keywords, identifiers, error messages, and comments must be in **Chinese**
  所有关键字、标识符、错误消息和注释必须使用**中文**
- Follow existing code style and module structure
  遵循现有代码风格和模块结构
- No third-party dependencies without prior discussion
  未经讨论不得引入第三方依赖

## What Counts as a Contribution / 什么算贡献

- Bug fixes / 缺陷修复
- Documentation improvements / 文档改进
- Test cases / 测试用例
- Performance optimizations / 性能优化
- New features (discuss first via Issues) / 新功能（请先通过 Issues 讨论）

## Contributor Recognition / 贡献者认可

Contributors are recognized in the following ways:
贡献者通过以下方式获得认可：

- **Git history**: Your name and contribution are permanently recorded
  in the project's Git commit history.
  **Git 历史**：您的姓名和贡献永久记录在项目的 Git 提交历史中。

- **Community acknowledgment**: Significant contributions may be
  acknowledged in project documentation or release notes.
  **社区致谢**：重要贡献可能在项目文档或发布说明中致谢。

- **Non-commercial license**: Contributors retain the right to use
  the project under the non-commercial terms of Nova-CSL 1.1.
  **非商业许可**：贡献者保留在 Nova-CSL 1.1 非商业条款下使用
  项目的权利。
